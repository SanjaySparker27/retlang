# Code of Conduct

This project follows the [Contributor Covenant v2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/).

By participating (issues, pull requests, discussions, chat), you agree to uphold its standards: be respectful, constructive, and patient.

## Reporting

If you witness or experience behavior that violates the Contributor Covenant, open a private security advisory on this repository or contact the maintainer via the email listed in `SECURITY.md`. Reports are handled privately.

## Enforcement

The maintainer may take any action deemed appropriate, including comment removal, temporary bans, or permanent bans from project spaces.
